## Installation

```
$ cd redmine/plugins
$ git clone https://github.com/fraoustin/redmine_indicator.git
$ rake redmine:plugins:migrate NAME=redmine_indicator
```

restart Redmine

for uninstall, you can use

```
$ cd redmine/plugins
$ rake redmine:plugins:migrate NAME=redmine_indicator VERSION=0
```
